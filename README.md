# NowRoad

**目前本项目已转移到新网站，知道过去旧网址的，请不要相关信息透露给他人**  

每周知识分享 **K** nowledge **S** hare **W<** eekly（KSW），是一个每周整理一期马列主义书籍、文章、视频，并写作推荐语的知识分享项目平台。项目内所有资源、材料与B站视频与项目管理者写作的推荐语均为合法，并未触犯任何中华人民共和国相关法律法规  

目前KSW下辖两个子项目。NowRoad旨在对马列主义 **历史爱好者** 对列宁主义革命路线的理解，通过描述、回顾列宁及其同志是怎样从分散的个体集结成组织，从脱离群众的书生到属于无产阶级的知识分子，回顾是怎样在工人中做长久牢固的宣传鼓动工作，是怎样与经济主义做斗争，是怎样办工人小组培养工人革命家，通过政治报统一思想凝聚力量，把各地分散的地方组织在实质上建成一个实质而非名义的先锋队的漫长历程，让读者理解那时的兴趣爱好者是怎样成为革命者，伊里奇怎样成为列宁，斗争协会是怎样成为社会民主工党，怎样成为布尔什维克并走上推翻政权的道路的。目前该子项目正在运行中。  

另一个子项目revealnow旨在回顾列宁等布尔什维克革命家们是如何在时事评论中揭露沙皇政权与资本主义的运行逻辑和本质，阐明唯一的出路方向，以增进无产阶级群众对现实的理解，目前该项目仍在筹备中，如果愿意提供投稿等支持的欢迎联系我们。
